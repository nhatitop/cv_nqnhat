HTML5

HTML or HyperText Markup Language, which is the most important component of the World Wide Web. It is the language used to describe what a Web site displays. However, if only HTML alone is quite boring because it can only provide static sites; To meet the growing demands for more impressive web features, HTML has been linked to plugins such as CSS, Flash, Java, Silverlight, etc.

CSS3

If you're building a house, tiles, rocks, sand, cement are HTML, and CSS is the formula, construction and layout that makes up the house. With ancient times people use clay instead of cement and sand, we called the ancient CSS, and this time we used cement and sand, it was CSS3.

JavaScript

JavaScript is the most popular programming language in the world during the past 20 years. It is also one of the 3 primary languages of HTML Web programming: Helping you add content to your website.
